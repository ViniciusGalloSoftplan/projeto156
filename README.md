# Projeto 156

Projeto web desenvolvido com HTML, CSS e JavaScript.

## Estrutura do Projeto

```
156/
├── index.html          # Página principal
├── categoria.html     # Página de categorias
├── assets/
│   ├── css/
│   │   └── style.css  # Estilos globais
│   └── js/
│       ├── main.js    # Script principal
│       ├── index.js   # Script da página inicial
│       ├── data.js    # Dados do projeto
│       ├── categoria.js # Script de categorias
│       └── setore.js  # Script de setores
└── README.md          # Este arquivo
```

## Funcionalidades

- Interface web responsiva
- Sistema de categorias
- Gerenciamento de dados

## Como Executar

1. Clone o repositório:
```bash
git clone https://github.com/ViniciusGalloSoftplan/projeto156.git
```

2. Abra o arquivo `index.html` no seu navegador
